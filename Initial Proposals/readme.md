# Game Design Proposals  

These are the 2 inital fleshed out game designs propsed to the team, which will be taking a vote on which to to move forward with. 

These games were drafted after discussing user stories, possible technical limitations, and a prioritized features list with the team. For example, one of our main priorities was ensuring the secondary genre of the game didn't introduce significant mechanicas that didn't involve typing. 

For these proposals, we focused on including the following characteristics: 
- **Character Design and Theme** - this would be significantly influenced by our user stories. We expected a younger audience, thus didn't want the graphics to be too intimidating, but still aimed to keep the game feeling serious.
- **Narrative** - the team had already decided on a general gameplay loop, where the user would type code syntax to discover clues, and solve a mystery, however the nature/format of those clues would be heavily reliant on the plot. Thus, a well plotted story was needed to provide to the other teams.
- **Tutorial Level** - Providing a tutorial level gave the team a concrete understanding of how the narrative may influence the other features. Additionally, our next upcomming task was to start on the tutorial level, so it proving this information upfront would ensure the other team members weren't reliant on us for further progression.
